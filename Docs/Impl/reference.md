# DenEmo 実装リファレンス

このドキュメントでは、DenEmo ワークスペース内の各ディレクトリおよびファイルの役割について説明します。

## ディレクトリ構造の概要

| ディレクトリ | 役割 |
| :--- | :--- |
| `Core/` | アニメーションの書き出し、プレビュー制御、対称性解析などのコアロジック。 |
| `Models/` | シェイプキーやアニメーションクリップを扱うデータモデルクラス。 |
| `UI/` | エディタウィンドウを構成する各セクションの UI 実装およびテーマ定義。 |
| `Utils/` | ローカライズ処理やプロジェクト設定の保存など、汎用的なユーティリティ。 |
| `Docs/` | ユーザーマニュアルや実装ドキュメント。 |
| `dennokoworks_color_schema/` | ツール全体で使用される共通のカラーデザインシステム。 |

---

## ファイル詳細

### ルートディレクトリ
- `DenEmoWindow.cs`: ツールのメインエントリポイントとなる EditorWindow クラス。全体のライフサイクルとメインのレイアウトを管理します。

### Core (コアロジック)
- `AnimationExporter.cs`: `AnimationClip` の生成、保存、およびメッシュへの適用ロジックを担当。
- `AnimationPreviewController.cs`: アニメーションのプレビュー再生およびキーフレームの記録・削除を制御。
- `LipSyncExclusionRule.cs`: VRChat 用の LipSync シェイプキーを自動的に除外するなどのフィルタリングルールを定義。
- `SymmetryParser.cs`: シェイプキー名の接頭辞・接尾辞（L/R など）を解析し、左右対称なペアを特定。

### Models (データモデル)
- `AnimationClipModel.cs`: 編集中のアニメーションクリップの状態（FPS、長さ、現在の再生時間など）を保持。
- `ShapeKeyItem.cs`: 個々のシェイプキーのデータ（名前、インデックス、現在の値、お気に入り状態など）を表すクラス。
- `ShapeKeyModel.cs`: 対象メッシュが持つ全シェイプキーのリストを管理し、検索やフィルタリングを提供。

### UI (ユーザーインターフェース)
- `AnimationModeUI.cs`: アニメーション編集モード時の UI ロジックと、タイムラインおよびプレビューの橋渡し。
- `AnimationTimelineUI.cs`: タイムライン、シークバー、再生コントロール、およびキーフレームトラックの描画。
- `ShapeKeyListUI.cs`: シェイプキーのリスト表示、スライダー操作、およびグループ化表示の管理。
- `DenEmoCommonUI.cs`: ヘッダーやステータスバーなど、ツール内で共通して使用される UI 要素。
- `DenEmoTheme.cs`: ツールのデザイン（配色、フォントスタイル、余白など）を定義する定数およびスタイル。

### Utils (ユーティリティ)
- `DenEmoLocalization.cs`: 多言語対応（日本語/英語）のためのテキスト管理と切り替え。
- `DenEmoProjectPrefs.cs`: `EditorPrefs` を使用した、プロジェクトごとの設定値の保存と読み込み。
